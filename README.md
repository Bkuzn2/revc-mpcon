# build:
1. [Download a NDK 28](https://developer.android.com/ndk/downloads)
2. Build a library 
> Win+R - cd C:/your folder path - C:\ndk\28.0.13004108\ndk-build.cmd
3. After library was built go to: "libs/armeabi-v7a" and "libs/arm64-v8a" 
4. Insert the libconnect.so in APK from sources
5. Install APK
